# Локальный редактор доски разрядов
Или же программа для локального создания тирлистов.

<img width="1920" height="991" alt="image" src="https://github.com/user-attachments/assets/0523146d-fa8a-459f-8a40-f188995a1128" />


Разработана и скомпилирована с помощью Codex, Cursor и Copilot с ручным переводом на русский язык. В основе своей лежит веб-страница с интерфейсом доступа через  pywebwiev. Можно искать игры, сортировать выдачу и загружать их на свою доску в виде карточки. Карточка содержит название игры, дату выхода, жанры, платформы, обложку и описание. Доска автоматически сохраняется локально в той же папке, где и программа.

Работа поиска основана на бесплатном RAWG API. Для возможности работы необходимо получить API ключ на сайте RAWG, создать в папке _internal файл .rawg_key, в который записать свой ключ.

Создано по подобию https://tiermaker.com/.
